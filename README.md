# Apple Tree Tots Preschool Homepage

A simple and elegant homepage for **Apple Tree Tots Preschool** created using **HTML** and **CSS**. This project serves as a professional landing page for the preschool, including sections for programs, admissions, parent portal, and contact information.

## Live Demo
Check out the live website here: [Apple Tree Tots](https://apple-tree-tots.vercel.app/)

## Features
- Fully structured homepage with navigation menu
- Sections for About Us, Programs, Admissions, Parent Portal, and Contact
- Parent login button (mock functionality)
- Clean and responsive design using HTML and CSS
- Easy to deploy using Vercel

## Technologies Used
- HTML5
- CSS3
- Vercel (for hosting)
